    说明：

    1. JS任务5-10统一采用萝卜多复盘项目接口
    2. 后台只有登录，无注册，注册在后台管理-角色管理中，任务5统一用提供的账密：admin   123456
    3. 学员环境：http://dev.admin.carrots.ptteng.com/;
    4. 跨域：
      学员服务器上的垮域已配置好，location拦截名为：
      后台——/carrots-admin-ajax/
      服务器nginx设置为：
      location /carrots-admin-ajax/{
      proxy_pass http://dev.admin.carrots.ptteng.com/;
      }

    字段
    说明

    备注

    pwd密码string

    字段
    说明
    类型
    备注

     message 说明信息 String 
     data 数据Object  

    data:

    字段
    说明
    类型
    备注
    manager用户信息Object  
     role 用户角色Object  

    manager:

    字段
    说明
    类型
    备注
    id用户IDNumber 
    name用户名称String 
    nick String 
    roleID角色IDNumber 
     status 状态 String 
     updateAt 更新时间 Number 
     updateBy 更新人 Number

     createBy 创建人 Number 

    role:
